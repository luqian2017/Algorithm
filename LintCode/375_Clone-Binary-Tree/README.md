375. Clone Binary Tree
中文English
For the given binary tree, return a deep copy of it.

Example
Example 1:

Input: {1,2,3,4,5}
Output: {1,2,3,4,5}
Explanation:
The binary tree is look like this:
     1
   /  \
  2    3
 / \
4   5
Example 2:

Input: {1,2,3}
Output: {1,2,3}
Explanation:
The binary tree is look like this:
   1
 /  \
2    3
Notice
You'd better not just return root to solve the problem.

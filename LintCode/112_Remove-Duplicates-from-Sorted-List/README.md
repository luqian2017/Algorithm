112. Remove Duplicates from Sorted List
中文English
Given a sorted linked list, delete all duplicates such that each element appear only once.

Example
Example 1:
	Input:  null
	Output: null


Example 2:
	Input:  1->1->2->null
	Output: 1->2->null
	
Example 3:
	Input:  1->1->2->3->3->null
	Output: 1->2->3->null

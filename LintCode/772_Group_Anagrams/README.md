772. Group Anagrams

Given an array of strings, group anagrams together.

Example
Example 1:

Input:
["eat","tea","tan","ate","nat","bat"]
Output:
[["ate","eat","tea"],
 ["bat"],
 ["nat","tan"]]
Example 2:

Input:
["eat","nowhere"]
Output:
[["eat"],
 ["nowhere"]]
Notice
All inputs will be in lower-case.
